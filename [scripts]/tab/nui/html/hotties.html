Server Hotkeys

Posted 3rd Dec 2017

Here we will keep an updated list of all the server Hotkeys:
F1 opens the phone menu
F2 opens the inventory menu
F3 opens the animations menu
F6 is used for some jobs see the job tuts for more info
F7 is the billing system for selling items
K is to set cruise control
U is to lock and unlock you car
E is the main action button for most things
M opens the tablet open up
BackSpace is used to close all menus
ESC or the home button can be used to close the tablet or head back to the home screen
Ctrl + R opens the radar when you are in a police vehicle
Chat Commands
/pv in the chat is spawn a car you have bought from the dealer
/ooc talk in out of character chat to the whole server
/twt make a tweet to the whole server
/unjail id **note jail commands only work when you are a police officer**
-->unjail someone
/jail1 playerid secsjailed
-->Put a person in jailcell 1
/jail2 playerid secsjailed
-->Put a person in jailcell 2
/jail3 playerid secsjailed
-->Put a person in jailcell 3
/repair is used to fix your car anytime and when your at the mechanics /repair fixes your vehicle completely

Server Features

Posted 3rd Dec 2017

Here we will keep an updated list of features and how to use them:
GAS: In the bottom right hand side of your map youll see a number with a percentage beside it, thats the gas gauge. If this number goes below ten percentage you'll notice your vehicle will drive weird. This means you need fuel. If you run out you can get a gas can from the station to fill your car or you can call the mechanic to help. To refill your vehicle head to a gas staion marked on the map and park in the orange marker.
TABLET: The tablet has everything you need to know about the server and more. We are constantly working on adding apps and features to it.
