# fxserver-esx_status
FXServer ESX Status Version+

####INSTALATION

1) CD in your resources/[esx] folder
2) Import esx_status.sql in your database
3) Add this into your server.scg
```
start esx_status
```
4) Delete the cache folder in your server-data
